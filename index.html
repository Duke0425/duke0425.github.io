Django基础配置(一)

创建一个写内容的项目,取名为
python manage.py startapp app

MVT:

  1. M: models.py,模型层:
  2. V: views.py, 写业务逻辑的地方
  3. T: templates,模板,定义页面

注意: MVT模式是由MVC模式演变出来的.

MVC模式是所有框架遵循的模式.

MVC:

  1. M: 模型层
  2. V: 模板,html页面
  3. C: 业务逻辑层 – controller 写业务逻辑的地方

一.settings.py中配置

DATABASES = {
'default': {
'ENGINE': 'django.db.backends.mysql',
'NAME': 'dj8',
'USER': 'root',
'PASSWORD': '123456',
'HOST': '127.0.0.1',
'PORT': 3306
}
}

如果出错,在DATABASE中添加’OPTIONS’:{‘isolation_level’:None}

二.安装pymysql

在工程目录的__init__.py文件中 定义 pymysql.install_as_MySQLdb

三.迁移和插入管理员账号

1. python manage.py migrate 迁移
2. python manage.py createsuperuser 插入管理员账号

四.如果在model.py中添加了表之后进行迁移

1. 生成迁移文件 python manage.py makemigrations
2. 迁移 python manage.py migrate

一.虚拟环境安装

第一步(安装virtualenv):

打开命令提示符(cmd)

pip install virtualenv /安装虚拟环境(pip install  --user virtualenv)

第二步(创建虚拟环境):
指定文件夹下:

pip virtualenv --no-site-packages -p C:\Program Files\Python37\python.exe djenv8

代码注解:
创建虚拟环境文件(–no-site-packages: 表示创建的虚拟环境是纯净的环境,不安装有其他的库)(-p : 表示虚拟环境中的环境参数+虚拟环境名字)

第三步 查看当前安装的库和库的版本

pip list

二.创建py项目

第一步(激活环境)

进入到先前创建虚拟环境的文件下,进入到scripts文件下,执行指令activate.即可激活环境.

windows: cd djenv8/Scripts 执行activate/ 用于激活环境  ---- (环境名)+路径 :表示成功激活

Mac/Linux/Ubuntu : 直接执行source activate命令

deactivate/退出虚拟环境

注意:激活activate后路径前面会括号,括号中有虚拟环境名

第二步(创建py)

在cmd中回到需要创建py项目的文件夹中,我的文件夹是在e盘下的workspace文件下.

django-admin startproject day01 / 创建py项目

使用pycharm打开创建的项目

在pycharm/setting/project/project interpreter中 右边齿轮点击add .

选择Existing environment ,添加先前创建在指定文件e:/env/djenv8 虚拟环境.ok退出,apply 使其生效即可.

启动:

打开Terminal:

启动指令: python manage.py runserver. 默认ip为127.0.0.1 默认端口是8000

修改启动端口 : python manage.py runserver 端口号

修改ip和端口: python manage.py runserver ip:端口号

ip参数:如果为0.0.0.0表示任何人都可以通过公网ip访问Django项目(80端口,隐藏)

redis 多种数据指令

2018-12-28

redis 多种数据类型指令

哈希表:

hset : 设置单个字段
hmset : 设置多个字段
hgetall : 获取全部字段和对应的值
hget : 获取字段和字段对应的值
hmget : 获取多个字段
hdel : 删除哈希表的字段

hkeys :查看全部字段
hvals :查看全部字段对应的值

列表:

(redis的列表可以实现两种经典的数据结构)

经典数据存储结构:
栈(stack) - FILO - 先进后出 lpush+lpop/ rpush+rpop
队列(queue) - FIFO - 先进先出 lpush+rpop/ rpush+lpop

lpush : 从列表左边插入值
rpush : 从列表右边插入值
lrange list start stop : 取出列表起始下标 - 停止下标的值
lpop : 从列表左边取值
rpop : 从列表右边取值

lindex : 获取对应的下标的值
llen : 获取列表元素个数
brpop :+等待时间

集合:

sadd :添加元素
smembers :查看列表的元素
srem :移出列表中的指定元素
sismember :查看元素是否存在于集合1
scard :查看所有的元素

sinter :交集
sunion :补集
sdiff :差集
spop :取元素并删除元素
sranmmember :随机从列表取出指定个数的元素

有序集合

zadd :添加元素 必须添加元素的score
zrange :从小到大按照score来排列元素
zrevrange :从大到小按照score来排列元素
