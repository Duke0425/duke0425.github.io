git远程仓库 ssh秘钥 服务器问题

##主体
关联方法1:
git clone 关联远程仓库
关联方法2:
git remote add origin 地址 关联远程仓库(只需关联一次)
git push -u origin master 第一次推送消息
git push 后面只用push.

拉取远程库的信息:
git pull

生成SSH 秘钥:
ssh-keygen
连接虚拟器:
ssh root@虚拟机公共ip
在本地传递文件到虚拟机上:
$ scp hah.rar root@120.79.219.2:/home/duke/temp.zip
启动nginx :
systemctl restart nginx
解压文件指令:

tar -xvf file.tar //解压 tar包

tar zxvf  MY_NAME.tar.gz 解压.tar.gz文件在当前目录

tar zxvf  MY_NAME.tgz 解压.tgz文件在当前目录
