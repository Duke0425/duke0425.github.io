redis 多种数据类型指令

哈希表:

hset : 设置单个字段
hmset : 设置多个字段
hgetall : 获取全部字段和对应的值
hget : 获取字段和字段对应的值
hmget : 获取多个字段
hdel : 删除哈希表的字段

hkeys :查看全部字段
hvals :查看全部字段对应的值

列表:

(redis的列表可以实现两种经典的数据结构)

经典数据存储结构:
栈(stack) - FILO - 先进后出 lpush+lpop/ rpush+rpop
队列(queue) - FIFO - 先进先出 lpush+rpop/ rpush+lpop

lpush : 从列表左边插入值
rpush : 从列表右边插入值
lrange list start stop : 取出列表起始下标 - 停止下标的值
lpop : 从列表左边取值
rpop : 从列表右边取值

lindex : 获取对应的下标的值
llen : 获取列表元素个数
brpop :+等待时间

集合:

sadd :添加元素
smembers :查看列表的元素
srem :移出列表中的指定元素
sismember :查看元素是否存在于集合1
scard :查看所有的元素

sinter :交集
sunion :补集
sdiff :差集
spop :取元素并删除元素
sranmmember :随机从列表取出指定个数的元素

有序集合

zadd :添加元素 必须添加元素的score
zrange :从小到大按照score来排列元素
zrevrange :从大到小按照score来排列元素
