redis基础

2018-12-27

一。linux操作redis

redis-server 前台运行redis ctrl + z :放置在后台 但是停止状态
redis-server & 后台运行redis
jobs - 查看后台进程.
bg %1 恢复后台进程.
fg %1 恢复前台运行
netstat -nap | grep 端口号 查看端口进程号
kill 进程号 : 杀死进程
kill -9 进程号: 强杀进程(一般不适用)

查询指令信息:

  1. man 指令
  2. 指令 –help
  3. info 指令:q

二.redis: 发展进程

lloogg.com

Github,Twitter,YouTube

提供高速缓存服务 - 缓存热点数据()
缓解了数据库的压力(高频访问数据不用直接访问数据库)

Redis提供两种数据持久化的方案

  • RDB
  • ADF

redis 操作指令:
auth – 验证身份 (输入密码)
ping –心跳事件
set username duke (ex time 添加超时时间) 设置key和值
get –查看键
del –删除键
expire :设置超时时间
keys * :查看所有的键
exist : 查看key是否存在.
flushdb: 删除当前库的所有键
flushall: 清除所有库的所有键
select (0-15) : 切换数据库,默认16库
save /bgsave - 保存数据/后台保存数据
quit : 退出
dbsize : 查看当前库有多少键值对
